# 🚀 GigTrack

**GigTrack** is a full-stack productivity application built as a Chrome Extension with a FastAPI backend.

It helps **job seekers** track job applications and **freelancers** manage proposals, clients, time tracking, and invoices—all from one place.

---

## ✨ Planned Features

### 👨‍💼 Job Seekers
- Save job postings with one click
- Track application status
- Application dashboard and analytics
- Follow-up reminders

### 💼 Freelancers
- Save freelance opportunities
- Track proposal status
- Client CRM
- Time tracking
- Invoice generation
- Earnings dashboard
